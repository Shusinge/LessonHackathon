# LessonHackathon
Hello, Ini adalah Ide saya :
Saya ingin membuat sebuah website edukasi yang bertujuan untuk menjelaskan
serta memberikan upaya pencegahan terhadap bencana alam akibat perubahan iklim,
seperti banjir, tanah longsor, kekeringan, dan cuaca ekstrem.

Website ini dibuat agar masyarakat, khususnya pelajar, dapat memahami penyebab,
dampak, serta langkah-langkah pencegahan dan penanganan bencana dengan bahasa yang
sederhana dan mudah dipahami.
